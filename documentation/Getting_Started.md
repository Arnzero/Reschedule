**GETTING STARTED**
npm init -y
"-y" pre-fills our package header info

**Dependencies installed**
npm install
    express
    express-ejs-layouts
    method-override
    mongoose


**Dev Dep installed**
npm install --save-dev
    dotenv
    nodemon

**Change our scripts in package.json**
"start": "node server.js",
    "devStart": "nodemon server.js"
